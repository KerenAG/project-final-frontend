# PredictApp - Sistema de Gestión de Proyectos con Análisis Predictivo

Proyecto Final de Carrera - ITLA 2025

## Descripción

PredictApp es una aplicación web diseñada para la gestión de proyectos de software con un enfoque en análisis predictivo. El sistema permite administrar proyectos, tareas y sprints, mientras que un módulo de inteligencia artificial analiza métricas para predecir posibles retrasos o riesgos en los proyectos.
